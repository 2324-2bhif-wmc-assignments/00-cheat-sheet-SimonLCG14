Lists

3 types of lists:

  • Unordered List: ul
  • Ordered List: ol
  • Describing list: dl

Unordered list

In an unordered list, the items are all equally ranked and the bullet point symbol is standardized a circle

Ordered list

In an ordered list, the items are listed with numbers

Describing list

In a describing list there is the description term <dt> and the description data <dd>

Styling for lists

You can style for example the bullet points in an unordered list:

  • Cool
  • Item
  1. Roman
  2. Oriented
  3. List
  4. Style

Properties for list-styling:

  • list-style

    Sets all properties in one declaration

  • list-style-image

    Specifies an image as the list item marker

  • list-style-position

    Specifies the position of the bullet points

  • list-style-type

    Specifies the type of list item marker

Tables

You can declare a table with the <table> tag

Tags for table:

  • <tr> table row
  • <thead> a table header
  • <th> table header cell
  • <tbody> table body
  • <td> table cell
  • <tfood> table footer
  • <td> table footer cell
